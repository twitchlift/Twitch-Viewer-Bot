# Contributing to TwitchLift

Thank you for your interest in contributing to **TwitchLift**, the #1 Twitch view bot and viewer bot service!

## How to Contribute

### Reporting Bugs
If you encounter any issues with the TwitchLift Twitch viewer bot service:

1. Check existing [issues](../../issues) to see if the bug has already been reported
2. Open a new issue using the **Bug Report** template
3. Include as much detail as possible (browser, plan, viewer count, screenshots)

### Suggesting Features
Have an idea to improve the TwitchLift view bot?

1. Check existing [issues](../../issues) for similar suggestions
2. Open a new issue using the **Feature Request** template
3. Describe the feature and how it would improve the viewer bot experience

### Documentation Improvements
Help us improve our docs:

1. Fork this repository
2. Make your changes in a feature branch
3. Submit a Pull Request with a clear description

### Code of Conduct
- Be respectful and constructive in all interactions
- Focus on improving the TwitchLift viewer bot experience for all users
- Follow the existing code style and documentation patterns
